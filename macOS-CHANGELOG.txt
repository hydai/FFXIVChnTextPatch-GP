# 簡介

對應原本 repo 的 [PatchTool 6.4 5/26 v3.1.0.2 版本](https://github.com/GpointChen/FFXIVChnTextPatch-GP/releases/tag/v3.1.0.2)
此處只提供 macOS 的版本使用，經測試 Intel 與 Apple Silicon 系列都能正確執行。

## 執行所需環境

使用前請確定你的環境有安裝 `openjdk`

我的版本為：
```
openjdk 11.0.18 2023-01-17
OpenJDK Runtime Environment Homebrew (build 11.0.18+0)
OpenJDK 64-Bit Server VM Homebrew (build 11.0.18+0, mixed mode)
```

推薦使用 [Homebrew](https://brew.sh/index_zh-tw) 進行安裝： `brew install openjdk@11`

## 執行步驟

1. 下載 FFXIVChnTextPatch-GP-macOS.zip
2. 解壓縮以後，點兩下 `FFXIVChnTextPatchGP` 執行即可

# 測試 app 版發布 (不需安裝 openjdk)

1. 下載 FFXIVChnTextPatch-GP-macOS-app.zip
2. 解壓縮以後，點兩下 `FFXIVChnTextPatchGP.app` 執行即可
